
# Supplier Price Watch - User Guide

## Introduction

Supplier Price Watch is designed to help businesses efficiently manage supplier price changes by automating the analysis of price lists and providing actionable insights. This guide will help you navigate the application and make the most of its features.

## Getting Started

### Creating Your Account

1. Visit [app.supplierpricewatchapp.com](https://app.supplierpricewatchapp.com)
2. Click "Sign Up" and enter your business email
3. Complete the registration form
4. Verify your email address
5. Set up your password

### First-time Setup

1. **Company Profile**: Enter your business information
2. **Connect Integrations** (optional):
   - Shopify store
   - Google Workspace
   - Gadget.dev account
3. **Upload Sample Data**: Upload your first supplier price list

## Core Features

### 1. File Upload and Analysis

#### Supported File Formats
- Microsoft Excel (.xlsx, .xls)
- PDF documents (.pdf)
- CSV files (.csv)

#### How to Upload Files
1. Navigate to the "File Upload" page
2. Click "Upload File" or drag and drop your file
3. Select file type (if not automatically detected)
4. Click "Analyze" to process the file

#### Understanding Analysis Results
- **Price Changes**: Items with price increases or decreases
- **New Items**: Products that weren't in previous price lists
- **Discontinued Items**: Products no longer available
- **Anomalies**: Unusual price changes that may require attention

### 2. Price Change Management

#### Filtering and Sorting
- Filter by price change percentage
- Filter by product category
- Sort by highest/lowest price change
- Filter by supplier
- Show only items with issues

#### Export Options
- Export to Excel
- Generate PDF report
- Share link with team members
- Export for Shopify import

### 3. Insights and Recommendations

#### AI-Generated Insights
- Impact assessment of price changes
- Suggested retail price adjustments
- Potential alternatives for high-increase items
- Market positioning analysis

#### Data Visualization
- Price change trends
- Supplier comparison charts
- Category breakdown of changes
- Margin impact visualization

### 4. Shopify Integration

#### Connecting to Shopify
1. Go to "Integrations" page
2. Click "Connect Shopify Store"
3. Enter your Shopify store URL
4. Authorize the application
5. Set up sync preferences

#### Syncing Price Changes
1. Select items to update
2. Review changes
3. Click "Sync to Shopify"
4. Confirm the update

### 5. Customer Notifications

#### Creating Notifications
1. Go to "Customer Notifications"
2. Select affected products
3. Set effective date for price changes
4. Customize notification template
5. Preview and send notifications

## Account Management

### User Roles and Permissions

| Role | Description | Capabilities |
|------|-------------|--------------|
| Admin | Full system access | All features, user management |
| Manager | Operational access | Analysis, sync, reports |
| Analyst | Read and analyze | Upload, analyze, export |
| Viewer | Read-only access | View analyses and reports |

### Managing Team Members

1. Go to "Settings" > "Team Management"
2. Click "Invite Team Member"
3. Enter email and select role
4. Customize access permissions
5. Send invitation

## Troubleshooting

### Common Issues

- **File Upload Errors**: Check file format and size limits
- **Shopify Connection Issues**: Verify API credentials and permissions
- **Sync Failures**: Check for product matching issues
- **Analysis Errors**: Ensure price list format is consistent

## Best Practices

### Optimizing Your Workflow

1. **Regular Updates**: Upload new price lists as soon as received
2. **Consistent Format**: Maintain consistent format in price lists
3. **Category Management**: Organize products by category for better insights
4. **Proactive Notifications**: Notify customers before price increases
5. **Data Validation**: Verify analysis results against original price lists

### Data Management

1. **Archiving**: Regularly archive old price lists
2. **Backup**: Export analyses for backup purposes
3. **Documentation**: Document supplier-specific nuances
4. **Cost Tracking**: Track cost changes over time for negotiation leverage
